# Tap Fight

抖音风格视频入口 × 上升平台 1v1 联机对战。

## 入口流程

```
打开 → 抖音风视频流（↑↓滑动）
     → 看视频自动匹主题 + BGM
     → ⚔️ 进入房间 → 对战
     → 结算 → 回视频流
```

## 玩法

- **1v1 联机**：近战（左）vs 远程（右），跨主题对战
- **上升平台**：平台从底部冒出上升，天花板/死亡地板碰即死
- **攻击**：近战 2 血、远程 1 血（1 秒冷却）、落石 1 血
- **特效**：斩击拖尾、命中火花、投掷物拖尾、奔跑灰尘
- **BGM**：每个主题独立背景音乐

## 技术栈

Canvas 2D + Vite + Socket.IO + Node.js

## 项目结构

```

├── client/
│   ├── index.html              # 视频入口 + 菜单 + 结算
│   ├── public/
│   │   ├── stage_config.json   # 场地配置
│   │   ├── videos/             # 5 个主题视频
│   │   ├── bgm/                # 2 个 BGM
│   │   └── theme_packs/        # 5 套主题素材
│   └── src/
│       ├── main.ts             # 全部逻辑（~1500 行）
│       └── styles.css
├── server/src/main.ts          # 房间管理 + 消息转发
└── theme_packs/                # 主题素材
```
